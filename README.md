# Claude Code ハンズオン資料 - 目次

## 📚 資料構成

このハンズオンは3つのドキュメントで構成されています：

### 1. **Quick Start ガイド** (`quick_start.md`)
**所要時間: 10分**

すぐに始められる実践的なガイドです。最短で Claude Code の高度な機能を体験できます。

**内容:**
- 10分でできる基本セットアップ
- すぐに使えるサンプルエージェント
- 即実行可能なカスタムコマンド
- 実用的なワークフロー例
- トラブルシューティング

**こんな人におすすめ:**
- とにかく早く試してみたい
- 実例を見ながら学びたい
- すぐに使えるテンプレートが欲しい

---

### 2. **メインハンズオン資料** (`claude_code_handson.md`)
**所要時間: 60-90分**

Claude Code の3つの高度な機能を体系的に学べる包括的なガイドです。

**学習内容:**
1. **サブエージェントの作成**
   - エージェントとは何か
   - 設定ファイルの構造
   - 専門分野特化型エージェントの作成
   - 使用方法とベストプラクティス

2. **出力スタイルのカスタマイズ**
   - スタイル設定の種類
   - フォーマット制御
   - 用途別スタイル設定
   - エージェントとの組み合わせ

3. **カスタムコマンドの作成**
   - コマンド定義の基本
   - パラメータ設計
   - フックの実装
   - ワークフロー自動化

**実践演習:**
- バイオインフォマティクス分析ワークフロー
- 設定ファイル一覧

**こんな人におすすめ:**
- 体系的に学習したい
- 各機能の詳細を理解したい
- 実際のプロジェクトに応用したい

---

### 3. **演習問題集** (`exercises.md`)
**所要時間: 3-5時間（演習による）**

実践的なスキルを身につけるための演習問題集です。

**演習内容:**
- **演習1**: サブエージェントの作成（3種類）
- **演習2**: 出力スタイルのカスタマイズ（3種類）
- **演習3**: カスタムコマンドの作成（3種類）
- **演習4**: フックの実装
- **演習5**: 統合プロジェクト（メタボロミクス解析）
- **演習6**: チーム設定の標準化
- **演習7**: CI/CD統合

**付録:**
- チートシート
- よく使うコマンド
- 解答例とフィードバック

**こんな人におすすめ:**
- 手を動かして学びたい
- 実践的なスキルを習得したい
- 自分のプロジェクトに応用する前に練習したい

---

## 🎯 学習パスの推奨

### パス1: 最速習得（30分）
```
Quick Start (10分) → メインハンズオン資料の流し読み (20分)
```
すぐに使い始めたい方向け

### パス2: 基礎固め（2-3時間）
```
Quick Start (10分) → メインハンズオン資料 (90分) → 演習1-3 (60分)
```
体系的に学びたい方向け

### パス3: 完全マスター（5-7時間）
```
Quick Start (10分) → メインハンズオン資料 (90分) → 全演習 (3-5時間)
```
プロジェクトで本格的に活用したい方向け

## 💡 使い方のヒント

### 初めての方
1. まず `quick_start.md` で実際に動かしてみる
2. 興味を持った機能を `claude_code_handson.md` で深く学ぶ
3. `exercises.md` で実践的なスキルを磨く

### 特定の機能だけ学びたい方
- サブエージェントだけ → `claude_code_handson.md` の該当セクション + 演習1
- 出力スタイルだけ → `claude_code_handson.md` の該当セクション + 演習2
- カスタムコマンドだけ → `claude_code_handson.md` の該当セクション + 演習3

### チームで導入する方
1. `quick_start.md` で全員が基本を体験
2. `claude_code_handson.md` でベストプラクティスを共有
3. `exercises.md` の演習6でチーム設定を標準化
